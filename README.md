# CMS Engine

Single-file, browser-based SOP (Standard Operating Procedure) editor with live preview.

## File

- `cms-engine-by-mvs.html` — canonical, self-contained editor (HTML/CSS/JS, no build step, no dependencies beyond Google Fonts CDN)

## Features

- Editable title and standard statement
- Per-section typography controls (font family, size, colour) for Title, Standard, Procedure, Checklist
- Block-based content: Procedure (ordered steps), Checklist (items w/ check state), Media (image + alt + caption)
- Block reordering and deletion
- Live-rendered preview card matching final output styling

## Usage

Open `cms-engine-by-mvs.html` directly in a browser. No server or build required.

## Status

Active — canonical file only. Do not create parallel/duplicate HTML files for this project.
